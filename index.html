신규사이트 구축

  • 작업 소개

    퍼블리싱팀 메인 업무로써, 솔루션 Admin을 이용하여 신규사이트를 구축하는 프로젝트입니다. 하단에 노출된 4개의 프로젝트들은 기여도가 높고 최근에 작업한 것, 완성도가 높은 편인 것들을 선별하여 게시하였으며, 그외의 프로젝트는 ALL PROJECT에서 찾아 보실 수 있습니다.

  • 작업 방식

    백앤드팀에서 개발한 솔루션 Admin으로 세팅된 개발 환경안에서, 디자인팀으로부터 전달받은 레이아웃을 퍼블리싱 합니다. 모든 신규 프로젝트는 반응형으로 제작이 되며, 디자인은 PC만 제공이 되어 모바일은 퍼블리싱팀에서 담당자의 재량껏 유동적으로 작업합니다.

  • 작업 기간

    일반적으로 회사소개 홈페이지 10페이지 이내의 소규모 프로젝트에서 퍼블리싱팀 1인이 맡아서 작업을 진행하며, 퍼블리싱은 대게 일주일 안에 완료됩니다. 다국어이거나 분량이 많은 경우 일정이 추가되거나 2인이 협업하여 작업이 진행되기도 합니다.

모듈 제작

  • 작업 소개

    일정이 비어있을 경우 진행하는 상시업무입니다. 클라이언트에게 제공될 섹션별 디자인 모듈을 반응형으로 미리 퍼블리싱 합니다. 모듈은 대략 100개 정도 작업하였습니다.

  • 작업 방식

    추후에 삽입될 텍스트나 이미지 등이 가변적이므로 다른 프로젝트들보다 더 유동적으로 CSS 스타일링하는데에 유의하여 작업합니다. SCSS, BEM, OOCSS를 구조적으로 작성하여 유지보수 및 업데이트를 용이하게 합니다. 개발자간의 약속을 통해 코드를 읽고 이해하는데 시간을 절약, 재사용을 높일 수 있도록 개발하였습니다.

  • 작업 기간

    일반적으로 스크립트가 필요없는 레이아웃은 모듈 한 모듈당 3시간 내외정도 소요됩니다.

갤러리형식으로 레이아웃을 저장해두었으며, 이미지 클릭시 html,css,script 코드가 나뉘어져있어서 모듈을 손쉽게 가져올 수 있습니다. 해당 작업은 회사 내부 비공개 프로젝트라 공개가 불가하여 notion 갤러리(초기에 진행했던 방식)로 예시를 대체합니다.

Tool/Language

  • 편해요
    • Photoshop

      고등학생시절, 싸이월드 손글씨만들기로 접하게 된 포토샵. 현재 1n년째 사용중

    • HTML5

      기본적인 마크업을 원활하게 가능하며 최근에는 SEO에도 관심을 가지기 시작했습니다

    • CSS3/SCSS

      대부분의 디자인 레이아웃 스타일링 가능. 협업을 위한 bem을 사용하게 되면서 scss를 활용하여 사용하고 있습니다

    • JQuery

      자유롭게 사용이 가능하여 포트폴리오 대부분을 제이쿼리 사용해서 작업하였습니다

    • Github

      전 회사에서 프로젝트 관리 Bitbucket을 통하여 개발자간 협업하며 작업해왔습니다

  • 어느정도 사용할 수 있어요
    • Javascript

      기본적인 코드 흐름을 파악할 수 있으며, 보통 코드를 읽고 수정하여 사용하는 정도로 사용하고 있습니다

    • XD

      Xd로 제작된 디자인 샘플 퍼블리싱 경험이 있으며 직접 툴을 사용하여 디자인 해 본 경험이 있습니다

    • Figma

      Figma로 제작된 디자인 샘플 퍼블리싱 경험이 있습니다

    • GSAP

      인터랙션 들어가는 사이트에서 사용 경험이 있습니다

    • Three.js

      현재 3d 인터랙션이 가장 트랜디하다고 생각하여 포트폴리오 제작하면서 시작해보게 되었습니다

신규사이트 구축

  • 작업 소개

    퍼블리싱팀 메인 업무로써, 사내 admin을 이용하여 신규사이트를 구축하는 프로젝트입니다.

  • 작업 방식

    백앤드팀에서 개발한 사내 Admin으로 세팅된 개발 환경안에서, 디자인팀으로부터 전달받은 레이아웃을 퍼블리싱 합니다. 모든 신규 프로젝트는 반응형으로 제작이 되며, 디자인은 PC만 제공이 되어 모바일은 퍼블리싱팀에서 담당자의 재량껏 유동적으로 작업합니다.

  • 작업 기간

    일반적으로 회사소개 홈페이지 10페이지 이내의 소규모 프로젝트에서는 퍼블리싱팀에서는 1인이 맡아서 작업을 진행하며, 퍼블리싱은 대게 일주일 안에 완료됩니다. 분량이 많은 경우 2인이 협업하여 작업합니다.

템플릿 제작

  • 작업 소개

    일정이 비어있을 경우 진행하는 상시업무로, 클라이언트에게 제공될 예시로 제작된 디자인 템플릿을 반응형으로 퍼블리싱 합니다.

  • 작업 방식

    추후에 삽입될 텍스트나 이미지 등이 가변적이므로 다른 프로젝트들보다 더 유동적으로 CSS 스타일링 합니다. 재사용성을 높히고, 개발자간 원활한 협업을 위해 BEM과 oocss를 혼용하여 scss를 작성합니다.

  • 작업 기간

    일반적으로 스크립트가 필요없는 레이아웃은 템플릿 한 템플릿당 3시간 내외정도 소요됩니다.

갤러리형식으로 레이아웃을 저장해두었으며, 클릭시 코드를 열람할 수 있습니다. notion을 사용하여 여러명이서 효율적으로 관리할 수 있습니다.
